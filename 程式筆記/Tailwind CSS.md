- [官方網站](https://tailwindcss.com/)
    
- 功能：
    
    - [[原子化 CSS]] 框架
        
    - 利用 class 快速設計 UI

- **特色**：
    
    - 提供大量預定義 class，例如 `p-4`、`text-center`、`bg-blue-500`
    - 支援響應式、狀態變化（如 `hover:`、`focus:`）、深色模式（`dark:`）
    - 可透過 `@apply` 合併 class，支援自定義主題與插件
    
    - **優勢**：
    - 社群活躍、文件完整
    - 與 Nuxt、Next.js、Vite 等現代框架整合良好
