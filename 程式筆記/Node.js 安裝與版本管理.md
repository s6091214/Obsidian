## 安裝 Node.js
- 直接到 [Node.js 官網](https://nodejs.org/) 下載
  - LTS（穩定版，推薦）
  - Current（最新功能）
- 建議用 **[[NVM 基本介紹與指令|nvm]] (Node Version Manager)** 來管理多版本
-   內建套件管理工具 **[[npm 基本介紹與指令|npm]]**

---

## 安裝與版本管理
- [Node.js 官網](https://nodejs.org/)
- 建議使用[[NVM 基本介紹與指令|nvm]] (Node Version Manager)** 管理版本

  ```bash
  nvm install 20
  nvm use 20
  node -v
  npm -v
```
