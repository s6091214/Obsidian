# 介紹

[間隔重複](https://www.youtube.com/watch?v=6H3uIAWJIfk)
## 概念
- 又稱 Spaced Repetition  
- 核心：把重複學習的時間間隔拉長，利用遺忘曲線來強化記憶  
- 目標：**用最少次數達到最佳記憶效果**

## 機制
- 遺忘曲線 (Ebbinghaus, 1885)  
  - 記憶會隨時間衰退  
  - 在快要忘掉之前複習，可以大幅延長記憶持久度  
- 漸進間隔：首次複習 → 1 天 → 3 天 → 7 天 → 15 天 …  

## 使用方法
- 將知識切成小單位 ([[抽認卡]] / 小筆記)  
- 設定重複間隔 (可以用 [[Anki]]、[[Quizlet]] 等工具)  
- 定期檢查自己是否真的記住，調整間隔長度
